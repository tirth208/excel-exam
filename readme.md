# 📊 Excel Dashboard Project

## 📌 Overview
This project contains an interactive Excel dashboard designed to analyze and visualize data efficiently. It helps users gain insights through charts, tables, and key performance indicators (KPIs).

## 📂 File Included
- `tirthvirani_exceldashbord.xlsx` – Main Excel dashboard file

## 🚀 Features
- Dynamic charts and graphs  
- Data filtering and sorting  
- User-friendly interface  
- KPI tracking  
- Automated calculations  

## 🛠️ Tools & Technologies
- Microsoft Excel  
- Pivot Tables  
- Charts & Graphs  
- Formulas & Functions  

## 📖 How to Use
1. Download the Excel file from this repository  
2. Open it in Microsoft Excel  
3. Enable editing and macros (if required)  
4. Use filters/slicers to interact with the dashboard
